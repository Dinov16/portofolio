Journey

  • Awal Perjalanan - Web Development

    Saya memulai pembelajaran dari pengembangan website dengan mempelajari dasar-dasar HTML, CSS, dan JavaScript.

  • Pendalaman Front-End dan Back-End

    Setelah memahami dasar web, saya melanjutkan ke front-end development, kemudian mempelajari back-end untuk memahami alur aplikasi secara menyeluruh. Seiring proses belajar, saya kembali memperdalam front-end agar kemampuan implementasi antarmuka tetap kuat.

  • Transisi ke Machine Learning

    Dari pengalaman pengembangan web, saya mulai beralih ke machine learning untuk mempelajari bagaimana data dapat digunakan dalam pengambilan keputusan cerdas.

  • Pendalaman Deep Learning

    Ketertarikan saya berkembang ke deep learning, khususnya dalam memahami model yang mampu menangani permasalahan kompleks secara lebih efektif.

  • Fokus Saat Ini - Computer Vision

    Saat ini saya berfokus pada machine learning dan computer vision, terutama pada image classification. Implementasi dan hasil penerapannya saya tampilkan pada menu Projects.
